CASA DE CAMBIO DIGITAL

Proyecto de casa de cambio digital. El objetivo es permitir al usuario
realizar intercambios de divisas entre dólar y tres criptomonedas: Bitcoin (BTC),
Ethereum (ETH) y Litecoin (LIT).

Para esto se utilizaron funciones que permiten realizar conversiones de valor,
aplicar comisiones y generar interacción directa con el usuario mediante cuadros de
diálogo (prompt, confirm, alert). También se emplearon estructuras condicionales
y ciclos para controlar el flujo de decisiones dentro del simulador.
